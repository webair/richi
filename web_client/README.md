# Richi Remote Door Web Client

## Project Setup

```sh
npm install
```

## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Compile and Minify for Production

```sh
npm run build
```

## Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
npm run lint-fix
```

## Check lint and type errors

```sh
npm run build
```
